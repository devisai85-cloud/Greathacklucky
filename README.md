# Simple Python Calculator

## Overview
This is a simple Python calculator project that supports basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features
- Addition
- Subtraction
- Multiplication
- Division

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/devisai85-cloud/Greathacklucky.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Greathacklucky
   ```

## Usage
You can run the calculator by executing the `calculator.py` script:
```bash
python calculator.py
```
